> 最終更新: 2026-04-03

## プロダクトビジョン

**「広告LP特化の高CVR比較サイト」**

リスティング広告からの流入ユーザーが、最短の意思決定フローでASPアフィリエイトリンクをクリックするよう設計する。SEOは二次的目標。CTAのクリック率最大化が最優先指標。

- ターゲットユーザー: 特定商材を検索・比較検討中のユーザー（購買意欲高）
- 収益モデル: ASPアフィリエイト報酬（成果報酬型）
- 展開商材: クレジットカード / 光回線 / 格安SIM / NISA・証券口座

## 機能要件

### 共通機能

#### グローバルナビ・カテゴリナビ
- ヘッダー: ロゴ + カテゴリリンク（クレカ/光回線/格安SIM/NISA）+ スティッキー固定（`position: sticky; top: 0`）
- カテゴリナビ: 各ページ上部にサブカテゴリタブ（例: クレカ → 年会費無料/高還元率/ゴールド/学生向け）
- モバイル: ハンバーガーメニューまたはスクロール横タブ
- PR表記バッジ: 商材ごとに `[PR]` / `[広告]` バッジを表示（景品表示法対応）

#### 比較表コンポーネント（ComparisonTable）
- テーブル形式（横スクロール対応、`min-width` 設定）+ グリッドカード形式の2表示モード
- ソート: 任意列のクリックで昇順/降順切替
- フィルタ: カテゴリ別条件でリアルタイム絞り込み（URLパラメータ反映）
- ランキング表示: 1位=金・2位=銀・3位=銅バッジ
- 行ホバー: ハイライト + CTA表示
- スティッキーヘッダー行: スクロール時もカラム名を固定

#### 詳細カードコンポーネント（ProductCard）
- 構成要素: ランク・商品名・タグ行・スペックグリッド（2列）・メリット/デメリット・おすすめ対象・CTAボタン
- タグ: 年会費無料/高還元/キャンペーン等をカラーバッジで表示
- メリット/デメリット: アイコン（+/-）付きリスト形式
- キャンペーン強調: ボーナス額を大文字・アクセントカラーで表示

#### CTAボタン（CTAButton）
- ページ内CTA配置: ページあたり最低15箇所（比較表行・カード・セクション末・フロート）
- リンク先: ASPアフィリエイトURL（外部リンク、`target="_blank" rel="noopener noreferrer"`）
- UTMパラメータ: `utm_source` / `utm_medium` / `utm_campaign` / `utm_content` を動的付与
- フロートCTA: モバイルのみ、ページ下部に常時表示（スクロール200px以降に出現）
- クリック計測: GTMカスタムイベント `cta_click` + `data-product` / `data-position` 属性

#### FAQアコーディオン（FAQAccordion）
- 各ページ最低8問を掲載
- 開閉アニメーション（CSS transition）
- 構造化データ（FAQPage JSON-LD）を出力

#### パンくず・内部リンク
- 構造化データ（BreadcrumbList JSON-LD）を出力
- 関連ページへの内部リンクをコンテンツ末尾に設置

#### PR/広告表記
- 各ページ冒頭に「本ページはPR・広告を含みます」の表記
- 商品カードごとに小さな `[PR]` バッジ
- ランキングロジックの説明文を設置（景品表示法・ステルスマーケティング規制対応）

#### 構造化データ（JSON-LD）
- `WebPage` / `BreadcrumbList` / `FAQPage` / `ItemList`（商品一覧）
- 各ページ `<head>` 内に `<script type="application/ld+json">` で出力

#### レスポンシブ（モバイルファースト）
- ブレークポイント: `sm` = 640px / `md` = 768px / `lg` = 1024px
- モバイル: 比較表 → 横スクロール + カードビューをデフォルト表示
- タップ可能要素: 最小44×44px（WCAG 2.5.5準拠）

#### Core Web Vitals対応
- 画像: `loading="lazy"` + WebP形式 + `width`/`height` 明示（CLS防止）
- フォント: `font-display: swap`
- CSS/JS: クリティカルCSSインライン化、非同期ロード

### 商材別機能

#### クレジットカード
- フィルタ項目: 年会費（無料/有料）/ ポイント還元率（〜0.5%/0.5〜1%/1%〜）/ ブランド（Visa/Mastercard/JCB/AMEX）/ 付帯保険
- ポイント計算ツール: 月間利用金額を入力 → 年間獲得ポイント数・相当金額を即時計算
- カテゴリ表示: 総合ランキング / 年会費無料ランキング / ゴールドカードランキング を並列表示
- キャンペーン金額の強調: ボーナスポイント額を大フォントで表示

#### 光回線
- 住居タイプ切替タブ: 戸建向け / マンション向け（ページ最上部）
- エリア検索: 都道府県セレクトボックス → 対応エリアのプロバイダを絞り込み
- 速度データ表示: 平均下り速度を横棒グラフで可視化（みんそく参考）
- 速度指標: 下り速度 / 上り速度 / Ping値 を表示
- キャッシュバック額の強調表示

#### 格安SIM
- 容量スライダー: 0〜50GB のレンジスライダー → 各プランの月額料金をリアルタイム更新
- 料金シミュレーター: データ容量 + 通話オプション → 月額合計を比較表形式で表示
- MNPガイド: 乗り換え手順をステップ形式で説明
- フィルタ: キャリア回線（docomo/au/SoftBank）/ 最低利用期間 / 5G対応

#### NISA・証券口座
- 手数料シミュレーター: 投資額・回数 → 年間手数料を比較計算
- 資産予測ツール: 月積立額・想定利回り・運用年数を入力 → 複利計算グラフ表示
- 口座開設フロー: 「開設の流れ」をステップカードで説明（最低3ステップ）
- 初心者向け導線: 「NISAとは？」「つみたてNISAとは？」の説明セクションを設置
- フィルタ: 口座タイプ（成長投資枠/つみたて枠/両対応）/ 手数料無料ETF対応

## 非機能要件

### パフォーマンス
| 指標 | 目標値 | 測定方法 |
|------|--------|---------|
| LCP（最大コンテンツ描画） | < 2.5秒 | PageSpeed Insights |
| FID（初回入力遅延）/ INP | < 100ms | Chrome UX Report |
| CLS（累積レイアウトシフト） | < 0.1 | PageSpeed Insights |
| TTI（インタラクティブまでの時間） | < 3.8秒 | Lighthouse |
| モバイルPageSpeed スコア | 80点以上 | Lighthouse |

### SEO
- 構造化データ: `WebPage` / `BreadcrumbList` / `FAQPage` / `ItemList`
- OGP: `og:title` / `og:description` / `og:image` / `og:url`
- Twitter Card: `summary_large_image`
- canonical URL: 全ページに `<link rel="canonical">` 設置
- サイトマップ: XML形式、ビルド時自動生成
- robots.txt: 適切なdisallow設定（管理画面等）
- hreflang: 日本語サイトのため `ja` のみ

### セキュリティ
- XSS対策: テンプレートエンジンによる自動エスケープ、`innerHTML` 直接操作禁止
- CSP（Content Security Policy）: HTTPヘッダーで設定
- 外部リンク: `rel="noopener noreferrer"` 必須
- フォーム（Phase 2）: CSRF対策 + reCAPTCHA

### アクセシビリティ（WCAG 2.1 AA準拠）
- 色コントラスト比: テキスト 4.5:1 以上、大テキスト 3:1 以上
- キーボード操作: 全インタラクション要素がTab/Enter操作可能
- スクリーンリーダー: `aria-label` / `role` 属性の適切な設定
- フォーカス表示: `focus-visible` スタイルの明示
- 代替テキスト: 全画像に `alt` 属性設定
